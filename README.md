# ai-marketplace

A collection of Claude Code plugins by Sebastien Martin.

## Setup

Add the marketplace in Claude Code:

```
/plugin marketplace add sebmartin/ai-marketplace
```

## Available plugins

### ai-workspace

Thread-based workspace management with specialized agents for long-running AI conversations.

```
/plugin install ai-workspace@sebmartin
```

### tech-expert-agents

Specialized expert agents for architecture, security, product strategy, technology selection, and cost analysis.

```
/plugin install tech-expert-agents@sebmartin
```
